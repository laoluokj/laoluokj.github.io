选择性必修一 高二月考试卷讲评（直线与圆 第二课时）教案

1. 教学目标

以高中数学核心素养为导向，学生能：① 通过三点坐标联立方程或一般式配方法求圆的标准方程，经历“设方程—列等式—解参数”的逻辑推理过程，提升数学运算素养；② 利用$d$（圆心到直线距离）与$r$（半径）的关系，分析直线与圆位置关系，求解弦长最小值、切线方程，发展直观想象与逻辑推理素养；③ 通过圆心距与半径和差的比较，判定圆与圆的位置关系，解决内含条件、切线四边形面积最小值问题，深化数学建模与运算素养；④ 在综合问题解决中体会直线与圆的内在联系，提升知识迁移与综合应用核心素养。

2. 教材分析

16、17题聚焦直线与圆的综合应用，涵盖圆的标准方程求解（三点式、一般式配方法）、直线方程（点斜式）、弦长计算（$|AB|=2\sqrt{r^2-d^2}$）、圆与圆位置关系（圆心距与半径和差比较）、切线方程（点到直线距离等于半径）、切线四边形面积（$S=2\sqrt{|QC_1|^2-r^2}$）等核心知识点，是对“直线与圆”内容的深化考查，注重方法应用与逻辑推导，体现选择性必修一的“综合化”设计特点。

3. 学情分析

普通高中普通班学生对直线与圆的基本概念有初步认知，但综合应用能力薄弱：求圆方程时联立方程易出错，弦长最小值不会转化为“圆心到直线距离的最大值”，圆与圆位置关系判定步骤不清晰，运算准确性有待提高，需加强基础方法总结与易错点突破。

4. 教学重点

  1. 圆的标准方程求解（三点式、一般式配方法）；
  2. 直线与圆位置关系的应用（弦长、切线）；
  3. 圆与圆位置关系的判定。

5. 教学难点

  1. 弦长最小值的推导（$|AB|=2\sqrt{r^2-d^2}$中$d$的最值）；
  2. 切线方程的求解（斜率存在性讨论）；
  3. 圆与圆内含条件的转化（$|C_1C_2|<|r_1-r_2|$）。

6. 教法

采用讲练结合法，结合几何画板直观演示，通过问题串引导思路，总结解题模板。

7. 学法

自主反思错题，小组讨论易错点，总结“求圆方程—析位置关系—算核心量”的解题步骤，提升运算准确性。

8. 教学准备

多媒体课件（含几何画板动态演示）、学生错题统计清单、草稿纸。

9. 课堂小结

  1. 求圆的标准方程有哪些方法？各自适用场景是什么？
  2. 利用$|AB|=2\sqrt{r^2-d^2}$求弦长最小值的关键是什么？
  3. 判定圆与圆位置关系的核心步骤是什么？

10. 教学反思

成功之处：通过错题统计精准定位学生问题（如16(1)联立方程符号错、17(3)漏斜率不存在情况），用几何画板演示“弦长随$d$变化”的过程，直观突破弦长最小值难点；总结“圆方程三步法”“圆与圆位置关系两步法”，帮助学生梳理思路。不足之处：部分学生运算错误（如17(1)配方法符号错）未逐一纠正，小组讨论时个别组效率低。改进设想：课前布置“错题归因”作业，课堂预留5分钟运算练习；明确小组讨论任务（如“分析16(2)②的推导逻辑”），派组长引导；课后发放“运算易错点清单”，加强巩固。

《高二月考试卷讲评 - 直线与圆 第二课时》教学设计

一、情景引入

同学们，生活中车轮滚滚向前，其轨迹蕴含着圆的奥秘，而道路交叉恰似直线相交。今天我们就从试卷中探寻直线与圆的奇妙关系。

  • 教师活动：通过生活中车轮和道路的例子，生动形象地引出本节课要讲评的直线与圆的内容。
  • 学生活动：认真倾听，联想生活中的直线与圆的实例，激发学习兴趣。
  • 设计意图：从生活实际出发，让学生感受到数学与生活的紧密联系，提高学生对本节课的关注度和学习积极性。

二、新知探究

探究活动一：圆的标准方程求解

  • 题目：已知点$M(1, -1)$、$N(2, 2)$、$P(3, 1)$，圆$C$经过$M$、$N$、$P$三点，求圆$C$的标准方程。
  • 教师活动：引导学生回顾圆的标准方程形式$(x - a)^2 + (y - b)^2 = r^2$，让学生思考如何利用已知的三个点来确定方程中的参数$a$、$b$、$r$。提示学生可以通过设方程，将三点坐标代入求解方程组。
  • 学生活动：积极思考，尝试设出圆的标准方程，将三点坐标代入方程，列出方程组$\begin{cases}(1 - a)^2 + (-1 - b)^2 = r^2 \ (2 - a)^2 + (2 - b)^2 = r^2 \ (3 - a)^2 + (1 - b)^2 = r^2\end{cases}$，并求解。
  • 设计意图：巩固学生对圆的标准方程的理解和运用，培养学生运用方程思想解决问题的能力，提高学生的计算能力和逻辑思维能力。

探究活动二：直线方程的求解

  • 题目：已知直线$l:x - 2y + 4 = 0$与直线$l_2:4x + 3y + 5 = 0$的交点为$P$。①若直线$l$经过点$P$，且斜率为$2$，求直线$l$的方程；②求过点$P$且在两坐标轴上的截距相等的直线方程。
  • 教师活动：首先引导学生联立直线$l$和$l_2$的方程$\begin{cases}x - 2y + 4 = 0 \ 4x + 3y + 5 = 0\end{cases}$，求出交点$P$的坐标。对于①，让学生根据点斜式方程$y - y_0 = k(x - x_0)$（其中$(x_0, y_0)$为直线上一点，$k$为斜率）来求解直线方程；对于②，提醒学生分截距为$0$和不为$0$两种情况讨论。
  • 学生活动：联立方程求解交点$P$的坐标，然后根据点斜式方程求出①中直线$l$的方程。对于②，分别讨论截距为$0$时设直线方程为$y = kx$，截距不为$0$时设直线方程为$\frac{x}{a} + \frac{y}{a} = 1$（$a
    eq0$），并求解。
  • 设计意图：强化学生对直线方程的不同形式的掌握和运用，培养学生分类讨论的数学思想，提高学生解决直线相关问题的能力。

探究活动三：圆与圆的位置关系判断

  • 题目：已知圆$C_1:x^2 + y^2 - 2mx + 4y + m^2 - 5 = 0$，圆$C_2:x^2 + y^2 + 2x = 0$，当$m = 5$时，圆$C_1$与圆$C_2$有什么位置关系？
  • 教师活动：引导学生将圆$C_1$和圆$C_2$的方程化为标准方程，求出两圆的圆心坐标和半径。然后让学生根据两圆的圆心距$d$与两圆半径$r_1$、$r_2$的关系（$d > r_1 + r_2$时，两圆相离；$d = r_1 + r_2$时，两圆外切；$|r_1 - r_2| < d < r_1 + r_2$时，两圆相交；$d = |r_1 - r_2|$时，两圆内切；$d < |r_1 - r_2|$时，两圆内含）来判断两圆的位置关系。
  • 学生活动：将圆$C_1$和圆$C_2$的方程化为标准方程，求出圆心坐标和半径。计算两圆的圆心距$d$，并与两圆半径进行比较，判断两圆的位置关系。
  • 设计意图：加深学生对圆与圆位置关系的理解和判断方法的掌握，培养学生运用几何知识解决问题的能力。

探究活动四：直线与圆相切问题

  • 题目：若直线$l$过点$P(1, 0)$，且与圆$C_1:x^2 + y^2 - 2mx + 4y + m^2 - 5 = 0$相切，求直线$l$的方程。
  • 教师活动：引导学生考虑直线斜率是否存在的情况。当直线斜率存在时，设直线方程为$y = k(x - 1)$，然后根据圆心到直线的距离等于半径（点到直线的距离公式$d = \frac{|Ax_0 + By_0 + C|}{\sqrt{A^2 + B^2}}$，其中$(x_0, y_0)$为圆心坐标，直线方程为$Ax + By + C = 0$）列出方程求解$k$；当直线斜率不存在时，直接判断直线是否与圆相切。
  • 学生活动：分情况讨论直线斜率是否存在，分别求出直线$l$的方程。
  • 设计意图：培养学生分类讨论的思想和运用点到直线距离公式解决直线与圆相切问题的能力。

探究活动五：四边形面积最小值问题

  • 题目：点$Q$在直线$y = -2x$上，过点$Q$作圆$C_1:x^2 + y^2 - 2mx + 4y + m^2 - 5 = 0$的两条切线，切点分别为$A$、$B$，求四边形$QACB$的面积的最小值。
  • 教师活动：引导学生分析四边形$QACB$的面积与切线长的关系，将四边形$QACB$的面积转化为两个三角形$QAC$和$QBC$的面积之和。然后让学生根据切线的性质，求出切线长与圆心到点$Q$的距离的关系，进而求出四边形面积的最小值。
  • 学生活动：思考四边形面积的转化方法，根据切线性质列出相关等式，求出四边形$QACB$面积的最小值。
  • 设计意图：提高学生综合运用直线与圆的知识解决问题的能力，培养学生的逻辑推理和转化思想。

三、课堂练习

  • 教师活动：给出类似试卷中的题目，如已知圆$C$的圆心在直线$x - y = 0$上，且圆$C$过点$(2, 2)$和$(0, -2)$，求圆$C$的标准方程；直线$l$过点$(1, 2)$，且与圆$x^2 + y^2 = 5$相切，求直线$l$的方程等。巡视学生做题情况，及时给予指导和纠正。
  • 学生活动：独立完成练习题，遇到问题主动向老师或同学请教。
  • 设计意图：通过课堂练习，让学生及时巩固所学的直线与圆的知识，提高学生运用知识解决问题的能力，同时教师可以了解学生对知识的掌握情况，以便进行有针对性的辅导。

四、课后练习

  1. 已知圆$C$的圆心在直线$2x - y = 0$上，且圆$C$与直线$x - y - 1 = 0$相切于点$P(2, 1)$，求圆$C$的标准方程。
  2. 已知圆$C_1:x^2 + y^2 - 4x - 6y + 9 = 0$，圆$C_2:x^2 + y^2 + 12x + 6y - 19 = 0$，判断两圆的位置关系。
  3. 直线$l$过点$(-1, 2)$，且与圆$(x - 1)^2 + y^2 = 4$相交于$A$、$B$两点，若$|AB| = 2\sqrt{3}$，求直线$l$的方程。
  • 教师活动：布置课后作业，提醒学生认真完成，强调作业的规范性和准确性。
  • 学生活动：按时完成课后作业，对课堂所学知识进行进一步的巩固和拓展。
  • 设计意图：通过课后练习，让学生在课后继续深入学习直线与圆的知识，加深对知识的理解和掌握，提高学生的综合解题能力。同时，教师可以通过批改作业了解学生的学习效果，为后续的教学提供参考。
