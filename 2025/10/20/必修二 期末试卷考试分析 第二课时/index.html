必修二 期末试卷考试分析 第二课时

Author: laoluokj Date: 十月 20, 2025  9:57:09

1. 教学目标

以高中数学核心素养为导向，达成以下目标：

  • 通过分析解三角形（题16）的条件（如(2c\cos B + b = 2a)），运用正余弦定理推导角(C)，发展逻辑推理素养；
  • 通过计算三角形周长、扇形面积（题17）及正三棱柱点面距离（题19），提升数学运算素养；
  • 通过扇形动点图形、正三棱柱空间结构的分析，强化直观想象素养；
  • 体会解三角形、三角函数、立体几何知识的应用价值，感悟“转化”（如等体积法）与“建模”（扇形面积模型）思想，落实数学建模素养。

2. 教材分析

北师大新版必修二聚焦“三角恒等变换与解三角形”“立体几何初步”核心内容。题16考查正余弦定理的条件适配与灵活应用；题17将三角函数与扇形面积结合，体现工具性；题19以正三棱柱为载体，强化空间几何转化思想（线面垂直→线线垂直、点面距离→等体积法）。三题覆盖模块核心，注重素养落地。

3. 学情分析

普通高中普通班学生，对解三角形公式记忆不牢、应用混淆（如题16易选错“边化角”或“角化边”）；扇形动点问题图形转化能力弱（题17难建立(\theta)与面积的关系）；立体几何空间想象与转化意识不足（题19不会用等体积法求点面距离），需分层引导与强化。

4. 教学重点

  1. 解三角形中正余弦定理的选择与应用（题16）；
  2. 立体几何中点面距离（等体积法）与线面垂直存在性分析（题19）；
  3. 扇形问题中三角函数模型的构建（题17）。

5. 教学难点

  1. 解三角形条件的公式适配（如题16选条件后的角(C)推导）；
  2. 立体几何中空间元素的转化（题19线面垂直→线线垂直的逻辑链）；
  3. 扇形动点问题的变量关系分析（题17中(\theta)与(S(\theta))的关联）。

6. 教法

采用“试卷反馈—典型精讲—方法归纳—变式巩固”的讲练结合法，结合多媒体动画直观演示与分层提问引导。

7. 学法

自主反思试卷错误原因，合作探究典型题解题思路，归纳核心方法，提升反思性学习与合作探究能力。

8. 教学准备

学生期末试卷、多媒体课件（扇形动点、正三棱柱动画）、正三棱柱模型、三角公式卡片。

9. 课堂小结

提出三个关键问题引导回顾：

  1. 解三角形时，给定“边+角”条件（如题16），如何选择“边化角”或“角化边”？
  2. 计算立体几何点面距离（题19），除等体积法外还有哪些方法？
  3. 解决“棱上是否存在点满足线面垂直”（题19（2）），核心思路是什么？

10. 教学反思

  • 成功之处：针对高频错误（如题16公式选错、题19等体积法遗忘）针对性讲解，结合模型强化空间想象，学生对“正余弦定理应用”“等体积法”的掌握率提升约30%。
  • 不足之处：部分基础弱的学生参与度低，题17的扇形变量分析停留于表面，互动集中在少数学生。
  • 改进设想：增加“小组错题分享”环节，用动画演示题17动点轨迹；对基础生布置“公式适配小练习”，强化正余弦定理的适用场景。

《必修二期末试卷考试分析 第二课时》教学设计

一、情景引入

同学们，在之前的考试中，三角函数、几何等知识是“重灾区”。比如盖房子确定角度、计算材料用量，就会用到这些知识。今天咱们就深入剖析相关考题。

教师活动

展示一些建筑、工程中运用三角函数和立体几何知识的图片，简要说明这些场景与本次要分析的题目相关，然后引出本节课要分析的考试题目。

学生活动

观看图片，聆听老师讲解，对本节课要分析的内容产生兴趣。

设计意图

通过生活实例引入，让学生明白数学知识在实际生活中的应用，提高学生对本节课的关注度和学习积极性。

二、新知探究

探究活动一：16 题分析

  1. 题目呈现
    在$\triangle ABC$中，内角$A$，$B$，$C$的对边分别为$a$，$b$，$c$，有以下三个条件：
    条件①：$2c\cos B + b = 2a$；
    条件②：$c\sin B - \sqrt{3}b\cos C = 0$；
    条件③：$(2a - b)\sin A + (2b - a)\sin B = 2c\sin C$。
    任选一个条件，解答下列问题：
    (1)求角$C$；
    (2)若$c = 2$，$\triangle ABC$的面积为$\sqrt{3}$，求$\triangle ABC$的周长。
  2. 分析过程
  • 选条件①
    • 由正弦定理将边化为角：$2\sin C\cos B + \sin B = 2\sin A$。
    • 因为$\sin A = \sin (B + C) = \sin B\cos C + \cos B\sin C$，所以$2\sin C\cos B + \sin B = 2(\sin B\cos C + \cos B\sin C)$。
    • 化简得：$\sin B = 2\sin B\cos C$，因为$\sin B
      eq 0$，所以$\cos C = \frac{1}{2}$，又$0 < C < \pi$，则$C = \frac{\pi}{3}$。
    • 已知$c = 2$，$S_{\triangle ABC} = \frac{1}{2}ab\sin C = \sqrt{3}$，可得$ab = 4$。
    • 由余弦定理$c^{2} = a^{2} + b^{2} - 2ab\cos C$，即$4 = a^{2} + b^{2} - ab = (a + b)^{2} - 3ab$，将$ab = 4$代入可得$(a + b)^{2} = 16$，所以$a + b = 4$，则$\triangle ABC$的周长为$a + b + c = 6$。
  • 选条件②
    • 由正弦定理得：$\sin C\sin B - \sqrt{3}\sin B\cos C = 0$，因为$\sin B
      eq 0$，所以$\tan C = \sqrt{3}$，又$0 < C < \pi$，则$C = \frac{\pi}{3}$。
    • 后续求周长过程同选条件①。
  • 选条件③
    • 由正弦定理得：$(2a - b)a + (2b - a)b = 2c^{2}$，即$a^{2} + b^{2} - c^{2} = ab$。
    • 根据余弦定理$\cos C = \frac{a^{2} + b^{2} - c^{2}}{2ab} = \frac{1}{2}$，又$0 < C < \pi$，所以$C = \frac{\pi}{3}$。
    • 后续求周长过程同选条件①。

教师活动

展示题目，引导学生回顾正弦定理、余弦定理以及三角形面积公式等相关知识。针对每个条件，逐步引导学生进行分析，在黑板上书写详细的推理过程，强调每一步的依据和思路。

学生活动

认真聆听老师讲解，回顾相关知识点，跟随老师的思路进行思考，在老师的引导下尝试自己推导一些步骤。

设计意图

通过对本题的分析，让学生熟练掌握正弦定理、余弦定理在解三角形中的应用，提高学生运用知识解决问题的能力，同时培养学生分类讨论的思想。

探究活动二：17 题分析

  1. 题目呈现
    如图，在扇形$OPQ$中，半径$OP = 1$，圆心角$\angle POQ = \frac{\pi}{2}$，$A$是扇形弧上的动点，$B$是半径$OP$所在直线上的动点，且$AB \perp OP$。记$\angle AOB = \theta$。
    (1)当$B$点与$P$点重合时，求$\theta$的值；
    (2)记$\triangle ABO$的面积为$S(\theta)$。
  2. 分析过程
  • (1)求$\theta$的值
    当$B$点与$P$点重合时，在$Rt\triangle AOP$中，$\cos\theta = \frac{OP}{OA} = 1$（因为$OA = OP = 1$），又$0 < \theta < \frac{\pi}{2}$，所以$\theta = \frac{\pi}{4}$。
  • (2)求$S(\theta)$
    在$Rt\triangle ABO$中，$OA = 1$，$OB = \cos\theta$，$AB = \sin\theta$，则$S(\theta) = \frac{1}{2}OB\cdot AB = \frac{1}{2}\cos\theta\sin\theta = \frac{1}{4}\sin 2\theta$，$\theta \in (0, \frac{\pi}{2})$。

教师活动

展示题目和图形，引导学生分析图形中的几何关系，提示学生利用三角函数的定义来解决问题。在讲解过程中，通过在图形上标注相关线段和角度，帮助学生理解。

学生活动

观察图形，思考老师提出的问题，尝试根据三角函数的定义来求解$\theta$的值和面积$S(\theta)$的表达式。

设计意图

培养学生的几何直观能力和运用三角函数知识解决几何问题的能力，让学生体会三角函数在实际几何图形中的应用。

探究活动三：19 题分析

  1. 题目呈现
    如图，在正三棱柱$ABC - A_{1}B_{1}C_{1}$中，$AB = AA_{1} = 2$，点$M$为$AB$的中点。
    (1)求点$A$到平面$MBC_{1}$的距离；
    (2)在棱$BB_{1}$上是否存在点$Q$，使得$AQ \perp$平面$BC_{1}M$？若存在，求出$\frac{BQ}{BB_{1}}$的值；若不存在，请说明理由。
  2. 分析过程
  • (1)求点$A$到平面$MBC_{1}$的距离
    • 首先，求出$\triangle MBC_{1}$的面积$S_{\triangle MBC_{1}}$和三棱锥$A - MBC_{1}$的体积$V_{A - MBC_{1}}$。
    • 以$\triangle ABC$为底面，$AA_{1}$为高，可先求出三棱柱的体积$V_{ABC - A_{1}B_{1}C_{1}}$，再通过分割法求出$V_{A - MBC_{1}}$。
    • 设点$A$到平面$MBC_{1}$的距离为$h$，根据$V_{A - MBC_{1}} = \frac{1}{3}S_{\triangle MBC_{1}}\cdot h$，进而求出$h$。
  • (2)判断是否存在点$Q$
    • 建立空间直角坐标系，以$AC$中点$O$为原点，$OB$，$OC$，$OO_{1}$所在直线分别为$x$，$y$，$z$轴。
    • 求出相关点的坐标，进而得到向量$\overrightarrow{AQ}$，$\overrightarrow{BC_{1}}$，$\overrightarrow{BM}$的坐标。
    • 若$AQ \perp$平面$BC_{1}M$，则$\overrightarrow{AQ} \cdot \overrightarrow{BC_{1}} = 0$且$\overrightarrow{AQ} \cdot \overrightarrow{BM} = 0$，通过解方程组判断是否存在满足条件的点$Q$，并求出$\frac{BQ}{BB_{1}}$的值。

教师活动

展示题目和图形，引导学生建立空间直角坐标系，讲解如何通过向量法解决立体几何中的垂直问题和距离问题。在黑板上详细书写坐标的计算过程和向量的运算过程。

学生活动

跟随老师的思路，建立空间直角坐标系，求出相关点的坐标和向量的坐标，尝试进行向量的运算和方程组的求解。

设计意图

让学生掌握利用空间向量解决立体几何问题的方法，提高学生的空间想象能力和运算能力，培养学生的逻辑推理能力。

三、课堂练习

教师活动

给出与本次分析题目类似的练习题，如：

  1. 在$\triangle ABC$中，内角$A$，$B$，$C$的对边分别为$a$，$b$，$c$，已知$a\cos B + b\cos A = 2c\cos C$，求角$C$。
  2. 如图，在扇形$OAB$中，半径$OA = 2$，圆心角$\angle AOB = \frac{\pi}{3}$，$C$是弧$AB$上的动点，$D$是$OA$所在直线上的动点，且$CD \perp OA$，记$\angle AOC = \alpha$，求$\triangle OCD$面积的最大值。
  3. 在正四棱柱$ABCD - A_{1}B_{1}C_{1}D_{1}$中，$AB = 2$，$AA_{1} = 4$，点$E$为$AB$的中点，求点$D$到平面$B_{1}CE$的距离。

巡视学生的做题情况，及时给予指导和纠正。

学生活动

独立完成练习题，遇到问题可以向老师或同学请教。

设计意图

通过课堂练习，让学生巩固所学的解题方法和技巧，及时反馈学生对知识的掌握情况，以便老师进行有针对性的辅导。

四、课后练习

布置课后作业，包括一些综合性的题目和拓展性的题目，如：

  1. 在$\triangle ABC$中，已知$3b = 2\sqrt{3}a\sin B$，且$\cos B = \cos C$，试判断$\triangle ABC$的形状。
  2. 如图，在圆锥$SO$中，底面半径$r = 1$，母线长$l = 2$，$M$是母线$SA$的中点，$N$是底面圆周上的动点，求点$M$到直线$SN$的距离的最小值。
  3. 在直三棱柱$ABC - A_{1}B_{1}C_{1}$中，$AB = AC = AA_{1} = 2$，$\angle BAC = 90^{\circ}$，点$D$为$BC$的中点，在棱$BB_{1}$上是否存在点$P$，使得$A_{1}P \perp$平面$ADC_{1}$？若存在，求出$BP$的长；若不存在，请说明理由。

教师活动

布置作业，强调作业的要求和完成时间，提醒学生认真对待作业，通过作业进一步巩固所学知识。

学生活动

按时完成课后作业，对作业中的问题进行思考和总结。

设计意图

通过课后作业，让学生进一步深化对知识的理解和应用，培养学生独立思考和解决问题的能力，同时为下节课的复习和总结做好准备。
