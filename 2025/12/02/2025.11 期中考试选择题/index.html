2025.11 期中考试选择题复习课教案

1. 教学目标

以高中数学核心素养为导向，引导学生：① 通过集合交集、函数奇偶性等概念的应用，深化数学抽象素养；② 借助直线与圆位置关系、解三角形的推理过程，发展逻辑推理素养；③ 准确完成平行直线距离、向量垂直（$\boldsymbol{m} \perp \boldsymbol{n} \Leftrightarrow 2(a-1) + a \cdot 1 = 0$）、解三角形（$a^2 = b^2 + c^2 - 2bc\cos A$）等运算，强化数学运算素养；④ 利用图形分析双曲线（$\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1$）、抛物线（$x^2 = 2py$）的几何特征，培养直观想象素养；⑤ 计算统计量（极差、平均数、分位数）并分析数据特征，提升数据分析素养。最终实现期中考试选择题解题能力与应试技巧的提升。

2. 教材分析

北师大新版教材注重“基础+综合”，试题覆盖集合、概率、直线与圆、函数、解三角形、圆锥曲线、向量、统计等模块，既考查概念本质（如集合交集、函数奇偶性），又考查方法迁移（如平行直线距离公式、圆的位置关系判断），契合教材“螺旋上升”的编排逻辑。

3. 学情分析

普通高中普通班学生，对基础概念（如圆锥曲线定义）的理解停留在表面，运算能力（如解三角形、向量垂直计算）薄弱，逻辑推理（如双曲线离心率推导）不足，需教师引导梳理知识点关联，强化解题步骤的规范性。

4. 教学重点

  1. 集合、函数奇偶性、直线与圆等基础知识点巩固；
  2. 解三角形、圆锥曲线等模块的解题方法总结。

5. 教学难点

  1. 双曲线、抛物线的综合几何应用；
  2. 复杂问题的逻辑推理与运算（如双曲线离心率推导）。

6. 教法

采用“讲练结合+分类指导+直观演示”法：结合实例讲解知识点，针对基础/综合题分层指导，用图形演示圆锥曲线几何特征。

7. 学法

学生自主思考完成题目，小组合作交流解题思路，总结错题原因，整理“题型-方法”对应清单。

8. 教学准备

多媒体课件（含试题解析、圆锥曲线图形演示）、学生答题情况统计分析表、错题集模板。

9. 课堂小结

  1. 本堂复习覆盖了哪些核心知识模块？
  2. 解决直线与圆、圆锥曲线问题的常用方法有哪些？
  3. 你在解题中最易出错的环节（如概念混淆、运算失误）是什么？

10. 教学反思

成功之处：以核心素养为线索串联知识点，讲练结合提升了学生对基础题（如集合、直线距离）的正确率；图形演示突破了双曲线离心率推导的直观想象难点。不足之处：部分学生对综合题（如双曲线离心率）的逻辑推理仍显吃力，课堂互动局限于“教师讲、学生听”。改进设想：增加分层练习（基础题+拓展题），针对不同层次学生设计任务；加强小组讨论，鼓励学生展示解题思路；课后针对圆锥曲线综合题开展1-2次专题辅导，强化推理过程的拆解。

高中数学期中考试选择题与填空题复习教学设计

一、情景引入

同学们，假如我们在生活中遇到很多数据集合、图形位置关系等问题，该如何解决呢？今天就通过这些考题来一探究竟。

  • 教师活动：通过简洁话语引出本节课围绕考试题目展开复习。
  • 学生活动：倾听并思考生活中可能遇到的相关数学问题。
  • 设计意图：以生活实例引发学生兴趣，自然过渡到本节课内容。

二、新知探究

探究活动一：集合与概率问题

  1. 题目呈现
    • 已知集合$A = {1, 2, 5}$，$B = {1, 2, 9}$，则$A \cap B =$（ ）
      • A.
      • B. ${1, 2}$
      • C. ${1}$
      • D. ${1, 2, 5, 9}$
    • 掷一颗质地均匀的正方体骰子，出现点数是$1$的概率是____。
  2. 教师活动
    • 展示题目，引导学生回顾集合交集的定义和概率的计算方法。
    • 提问学生集合交集的概念，以及计算骰子出现特定点数概率的思路。
  3. 学生活动
    • 思考并回答教师的问题。
    • 尝试计算集合的交集和骰子出现点数$1$的概率。
  4. 设计意图
    • 巩固集合交集和简单概率的基础知识，为后续更复杂的题目做铺垫。
    • 培养学生回顾和运用旧知识解决问题的能力。

探究活动二：直线与圆的问题

  1. 题目呈现
    • 两条平行直线$l_1: x - y - 2 = 0$与$l_2: x - y = 0$之间的距离为（ ）
      • A. $2$
      • B. $1$
      • C. $\frac{\sqrt{2}}{2}$
      • D. $2$
    • 若方程$x^2 + y^2 + 2x - a = 0$表示的曲线是一个圆，则实数$a$的取值范围是（ ）
      • A. $(-\infty, 1)$
      • B. $(1, +\infty)$
      • C. $(-\infty, -1)$
      • D. $(-1, +\infty)$
    • 圆$C_1: x^2 + (y - 4)^2 = 4$与圆$C_2: (x - 3)^2 + y^2 = 9$的位置关系是（ ）
      • A. 内含
      • B. 外切
      • C. 内切
      • D. 相交
  2. 教师活动
    • 讲解直线平行距离公式、圆的一般方程的条件以及判断两圆位置关系的方法。
    • 引导学生分析题目，找出解题的关键步骤。
    • 针对学生的回答进行点评和纠正。
  3. 学生活动
    • 认真听讲，理解教师讲解的知识点。
    • 积极思考题目，尝试运用所学知识解题。
    • 与同学交流讨论，分享自己的解题思路。
  4. 设计意图
    • 加深学生对直线与圆相关知识的理解和运用。
    • 培养学生分析问题和解决问题的能力，以及合作交流的意识。

探究活动三：函数与三角函数问题

  1. 题目呈现
    • 已知$f(x)$是定义在$R$上的偶函数，且当$x < 0$时$f(x) = x + 2$，则$f(2) =$（ ）
      • A. $0$
      • B. $-4$
      • C.
      • D. $-2$
    • 在$\triangle ABC$中，角$A$，$B$，$C$的对边分别是$a$，$b$，$c$，若$b = 1$，$c = 3$，$\cos A = \frac{1}{3}$（假设补充完整），则$a =$（ ）
      • A. $4$
      • B. $2\sqrt{2}$
      • C. $2\sqrt{3}$
      • D. $\sqrt{14}$
  2. 教师活动
    • 强调偶函数的性质，引导学生利用偶函数的对称性解题。
    • 讲解余弦定理的内容和应用，通过题目示例让学生掌握如何运用余弦定理求三角形的边长。
    • 鼓励学生提出疑问，及时解答学生的困惑。
  3. 学生活动
    • 回顾偶函数的性质，思考如何利用已知条件求出$f(2)$的值。
    • 学习余弦定理，尝试运用余弦定理解决三角形边长的问题。
    • 提出自己在解题过程中遇到的问题，与教师和同学共同探讨。
  4. 设计意图
    • 强化学生对函数奇偶性和三角函数余弦定理的理解和运用。
    • 培养学生独立思考和解决问题的能力，以及提出问题和解决问题的意识。

探究活动四：双曲线与抛物线问题

  1. 题目呈现
    • 如图，双曲线$C: \frac{x^2}{b^2} - \frac{y^2}{a^2} = 1$（$a > 0$，$b > 0$）的左、右焦点分别为$F_1$，$F_2$，$O$为原点，在$C$的右支上存在关于$x$轴对称的两点$P$，$Q$，使得$\triangle PF_1Q$为正三角形，且$OQ \perp FP$，直线$OQ$交$FP$于点$M$，则双曲线$C$的离心率为（ ）
      • A. $2$
      • B. $1 + \sqrt{2}$
      • C. $3$
      • D. $1 + \sqrt{3}$
    • 已知抛物线$C: x^2 = 2py$的焦点为$F$，$P$为其上一动点，当$P$运动到$(t, 1)$时，$|PF| = 2$，直线$l$与抛物线$C$相交于$A$、$B$两点，则下列结论正确的是（ ）
      • A. 抛物线$C$的方程为$x^2 = 4y$
      • B. 抛物线$C$的准线方程为$y = -1$
      • C. $|PF| > 1$
      • D. 当直线$l$过焦点$F$时，以$AF$为直径的圆与$x$轴相切
  2. 教师活动
    • 详细讲解双曲线离心率的定义和求法，结合题目中的几何条件，引导学生建立等式求解离心率。
    • 介绍抛物线的焦点、准线等概念，以及抛物线的定义和性质。
    • 通过图形辅助，帮助学生理解题目中的几何关系，引导学生分析解题思路。
  3. 学生活动
    • 认真听讲，理解双曲线离心率和抛物线相关概念。
    • 观察图形，分析题目中的几何条件，尝试找出解题的关键。
    • 与教师和同学互动，分享自己的解题思路和遇到的困难。
  4. 设计意图
    • 提升学生对双曲线和抛物线这两种圆锥曲线的综合运用能力。
    • 培养学生的几何直观和逻辑推理能力，提高学生解决复杂数学问题的能力。

探究活动五：多项选择题与数据统计问题

  1. 题目呈现
    • 二选择题：本题共$3$小题，每小题$6$分，共$18$分。在每小题给出的选项中，有多项符合题目要求。全部选对的得$6$分，部分选对的得$3$分，有选错的得$0$分。
    • 关于直线$l: 3x + y + 2 = 0$，下列说法正确的有（ ）
      • A. 直线$l$的斜率为$-3$
      • B. 直线$l$的倾斜角为$120^{\circ}$
      • C. 直线$l$在$y$轴上的截距为$-2$
      • D. 直线$l$不经过第一象限
    • 为了弘扬航天人顽强拼搏的精神，某校航天课外小组举行了一次航天知识竞赛，随机抽取$6$名同学的分数（满分$30$分）：$20$，$22$，$24$，$24$，$26$，$28$，则这组数据的（ ）
      • A. 极差为$8$
      • B. 平均数为$24$
      • C. $80%$分位数为$25$
      • D. 众数为$24$
  2. 教师活动
    • 强调多项选择题的得分规则，提醒学生仔细分析每个选项。
    • 引导学生回顾直线的斜率、倾斜角、截距等概念，以及数据统计中的极差、平均数、分位数和众数的计算方法。
    • 组织学生讨论多项选择题的解题策略，鼓励学生分享自己的思考过程。
  3. 学生活动
    • 了解多项选择题的得分规则，认真分析每个选项。
    • 回顾直线和数据统计的相关知识，运用所学知识判断选项的正确性。
    • 参与小组讨论，分享自己的解题思路和答案，听取他人的意见和建议。
  4. 设计意图
    • 培养学生的综合分析能力和严谨的思维习惯，提高多项选择题的得分率。
    • 巩固直线和数据统计的基础知识，提升学生运用数学知识解决实际问题的能力。

探究活动六：填空题

  1. 题目呈现
    • 已知平面向量$\vec{m} = (2, a)$，$\vec{n} = (a - 1, 1)$，若$\vec{m} \perp \vec{n}$，则实数$a$的值为____。
    • 若椭圆$C: \frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$（$a > b > 0$）的离心率为$e$，且$e = \frac{\sqrt{3}}{2}$，则$b =$（ ）
  2. 教师活动
    • 提醒学生填空题的答题规范，强调计算的准确性。
    • 引导学生回顾向量垂直的性质和椭圆离心率的公式。
    • 对学生的解答进行点评和总结，强调解题的关键步骤和易错点。
  3. 学生活动
    • 认真读题，注意答题规范。
    • 运用向量垂直的性质和椭圆离心率的公式进行计算。
    • 检查自己的答案，确保计算的准确性。
  4. 设计意图
    • 巩固向量垂直和椭圆离心率的相关知识，提高学生的计算能力和答题规范。
    • 培养学生严谨的数学思维和认真审题的习惯。

三、课堂练习

  1. 教师活动
    • 给出一些与上述探究活动类似的练习题，要求学生独立完成。
    • 巡视学生的做题情况，及时发现学生存在的问题并给予指导。
    • 选择部分学生的答案进行展示和讲解，强调解题的思路和方法。
  2. 学生活动
    • 独立完成练习题，运用所学知识解决问题。
    • 遇到问题时，先尝试自己思考解决，若无法解决则向教师或同学请教。
    • 认真聆听教师对练习题的讲解，对比自己的解题思路，总结经验教训。
  3. 设计意图
    • 及时巩固课堂所学知识，提高学生运用知识解决问题的能力。
    • 发现学生在学习过程中存在的问题，以便及时进行针对性的辅导和强化。

四、课后练习

  1. 题目布置
    • 布置适量的课后作业，包括选择题、填空题和解答题，涵盖本节课复习的所有知识点。
    • 要求学生认真完成作业，注意答题规范和书写工整。
  2. 教师活动
    • 批改学生的课后作业，了解学生对本节课知识的掌握情况。
    • 针对学生作业中出现的问题，进行集中讲解和个别辅导。
  3. 学生活动
    • 按时完成课后作业，独立思考，认真书写。
    • 对作业中出现的错误进行反思和总结，分析原因并加以改正。
  4. 设计意图
    • 进一步巩固课堂所学知识，加深学生对知识点的理解和运用。
    • 通过作业反馈，了解学生的学习情况，为后续的教学提供参考。
